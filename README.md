> * ML의 **4가지 분류**
> 
> - Vision: 컴퓨터 비전을 사용하여 이미지와 비디오를 처리하고 분석
> - Natural Language: 단어 포함, 분류와 같은 다양한 방법으로 텍스트를 처리하고 이해
> - Speech: 다양한 언어에 대한 음성 인식 및 현저성 기능을 활용
> - Sound: 오디오를 분석하고 웃음이나 박수같은 특정 유형으로 분류/인식
